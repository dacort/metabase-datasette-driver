## Installation

(Requires Metabase v0.35)

- Drop the jar into your Metabase `plugins/` directory. 
- Restart Metabase
- `Datasette` should now show up in your list of available databases.
- Add an endpoint (e.g. https://latest.datasette.io/fixtures)
